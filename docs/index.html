Product Modules

五个模块串起完整养宠流程

宠物档案

记录名称、年龄、品种、体重、健康状态，让后续健康和行为数据拥有统一入口。

宠物健康

支持健康记录、体重体温追踪、定时喂食提醒和模拟远程投喂。

宠物行为

围绕活动、睡眠、情绪和位置关系建立日常观察面板。

宠物领养

浏览待领养信息、提交申请，并将领养结果衔接到宠物档案。

电子萌宠

通过文本与实时语音和“奶糖”交流，并用基础记忆延续上下文。

AI Companion

奶糖，让工具变成陪伴入口

Architecture

纯北向开发，突出 HarmonyOS 应用能力

ArkTS / ArkUI

原生界面与状态刷新

采用 ArkUI 组件化搭建页面，用轻量级消息总线处理跨组件数据变化后的同步刷新。

Node.js / SQLite

后端服务与数据存储

通过后端接口和本地数据库完成宠物档案、健康、行为、日历等模块的数据管理。

Multi Device

双设备同步

使用时间戳版本判断与删除标记处理覆盖、删除和二次上传冲突。

Responsive

折叠屏适配

根据屏幕宽度切换单栏与双栏布局，展开过程保持当前操作上下文。
